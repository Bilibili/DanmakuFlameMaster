DanmakuFlameMaster
==================

android上的弹幕转换、渲染输出源码


### Features

- 使用SurfaceView高效绘制

- A站json弹幕格式解析

- B站xml弹幕格式解析

- 基础弹幕精确还原绘制

- 支持特殊弹幕

- 多核机型优化，可选独立线程缓存进行绘制

- 支持多种绘制选项动态切换

- 支持实时弹幕显示

- 换行弹幕支持/运动弹幕支持

- 支持自定义字体

- 支持多种参数设置
